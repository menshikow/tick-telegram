```md
A minimal telegram bot for managing your daily-life tasks
```

---

## Commands

- `/add <title> [description]`
- `/list`
- `/done <index>`
- `/adddesc <index> <description>`
- `/delete <index>`
- `/clear`
- `/help`

## Localization

- English (default)
- Russian — automatically selected when Telegram is set to Russian
